# AL-SAMAD GLOBAL TRADER

Official website for AL-SAMAD GLOBAL TRADER — Exporter, Bulk Supplier & Global Trade Partner based in Vijayawada, Andhra Pradesh, India.

## Tech Stack

- **Frontend:** React + Vite, Tailwind CSS, shadcn/ui
- **Backend:** Node.js, Express

## Project Structure
    apps/

├── api/     # Express backend

└── web/     # React frontend

## Getting Started

### Frontend
```bash
cd apps/web
npm install
npm run dev
```

### Backend
```bash
cd apps/api
npm install
node src/main.js
```

## Features

- Home, About, Products, Services, Markets, Blog, Contact pages
- Responsive design
- Contact form with backend API integration
